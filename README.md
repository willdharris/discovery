# Discovery EPG

A web app to view the programming schedules for 16 different Discovery networks all in one place.

- [Demo](https://discoveryepg.netlify.app/)

Built with:

- HTML
- CSS
- Vanilla JavaScript
- REST API

Schedules sourced from:

- [Schedules Direct API](https://www.schedulesdirect.org/)

## How it works

- Each grid contains 4 days of programming info for a network.
- Navigate to a network grid and scroll to desired date and time to view scheduled programming.
- Each grid can be expanded to view more detailed info, including season, episode, and Gracenote IDs where available.
